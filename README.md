# Mica Portfolio (Streamlit Multipage App)

A modern personal portfolio web application built using Python Streamlit.  
This project showcases my profile, skills, projects, certificates, and contact information in a clean and interactive multipage layout.

---

## Features

Home / About Me page with personal introduction  
Skills section with progress indicators  
Projects showcase using interactive expander cards  
Certificates gallery with hover effects and animations  
Contact form with validation and responsive layout  
Modern glassmorphism UI design  
Animated gradient background consistent across all pages  
Fully responsive design for mobile, tablet, and desktop devices  

---

## Built With

Python  
Streamlit  
HTML and CSS (custom styling inside Streamlit)  
Base64 image encoding for certificates
